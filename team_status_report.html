4보완 사항 및 과제

샘플 크기 한계 (n=125) 중요
21개사×6년이라 일부 회귀계수의 p값이 여전히 높음(유의하지 않음). 발표 시 "방향성 참고용"이라는 점을 명시하고, 가능하면 업종 추가 확장(타이어사 등) 고려.
이자보상배율·영업이익률 예측력 낮음 (R² 0.18~0.24) 중요
거시변수만으로는 설명력 부족. 기업 개별 요인(업종 내 포지션, 고객사 의존도 등) 피처 추가하면 개선 가능. 시간 부족하면 "거시변수의 한계"로 리포트에 솔직히 명시.
지속성 기준선 대비 예측력 보완 중요
관측범위 가드레일은 구현 완료. 다만 동일 표본에서 직전 연도·분기 값을 그대로 쓰는 기준선이 대부분의 XGBoost보다 높으므로 현재 모델은 예측기보다 민감도·설명 도구로 제한.
명신산업·캐프·화신테크 데이터 결측 보통
상장 시기가 늦어 일부 연도 데이터 없음(n_quarters<4). 현재 코드는 경고만 출력하며 패널에는 포함됨. 리포트에서 해당 기업 선택시 "데이터 제한적" 안내 필요.
실제 위험 결과변수 검증 부재 중요
Gemini 연동과 수치보존 프롬프트는 구현 완료. 현재 목표값은 재무비율 자체이므로 부도·등급하락·감사의견 같은 외부 사건을 예측했다고 해석할 수 없음. 향후 결과변수 라벨 검증 필요.
실제 Cloud 배포 낮음
PDF·README·배포 설정·5장 발표자료는 완료. 사용자가 배포를 결정하면 GitHub와 Streamlit Cloud를 연결.
타이어 3사(넥센·한국타이어·금호타이어) 포함 여부 낮음
처음 범위 정의 시 별도 업종으로 분류해 제외했음. 샘플 보강이 필요하면 추가 검토 가능하나 2주 일정상 우선순위는 낮음.

△ 예측력 — 지속성 기준선 우세

교정 분기 Z-score 모델 R²는 0.902지만 직전 분기 기준선은 0.934. 대부분 지표에서 단순 기준선이 우세하며, 분기 유동비율만 0.818로 기준선 0.793을 소폭 상회.

△ 금리 연관성 — 반복 관측, 인과 아님

Z-score↔한국금리는 교정 연간 p=0.005, 교정 분기 p=0.002. 다만 거시변수는 같은 시점의 모든 기업에 반복되고 시간 관측치가 제한적이므로 인과효과로 해석하지 않음.
