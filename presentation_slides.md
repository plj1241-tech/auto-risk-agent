# 자동차 부품업계 리스크 분석 에이전트

공시 재무데이터와 거시지표를 결합한 **위험 모니터링·설명 프로토타입**

- 대상: 자동차 부품 상장사 21개
- 기간: 2019~2024년
- 데이터: DART · ECOS · FRED
- 기능: 현재 위험, 동종 비교, SHAP 설명, 관측범위 내 민감도, 리포트

> 미래 부도·주가를 예측하는 투자모델이 아니라 재무 위험 점검을 돕는 분석 도구입니다.

---

# 1. 데이터와 파이프라인

- 연간 패널: **125행 × 32열**, 기업·연도 중복 0건
- 분기 패널: **492행 × 36열**, 21개사
- 위험지표: 부채비율, 유동비율, 이자보상배율, 영업이익률, Altman Z-score
- 거시지표: 한국금리, 환율, 미국금리, PPI, WTI, 철광석

## 데이터 교정

- DART Q1~Q3 `thstrm_amount`는 해당 분기값으로 유지
- Q4 연간값만 Q1~Q3을 차감해 단일분기로 변환
- 음수 매출 **102건 → 2건**
- 자본잠식 기업의 음수 부채비율은 결측 처리 후 별도 경고

---

# 2. 분석 결과 — 무엇이 유의미한가

| 지표 | 연간 XGB R² | 직전연도 기준선 | 판정 |
|---|---:|---:|---|
| 부채비율 | 0.554 | 0.854 | 기준선 미달 |
| 유동비율 | 0.858 | 0.913 | 기준선 미달 |
| 이자보상배율 | 0.240 | 0.286 | 둘 다 낮음 |
| 영업이익률 | 0.242 | 0.514 | 기준선 미달 |
| Z-score | 0.794 | 0.905 | 기준선 미달 |

- 기업별 재무 수준과 지속성 신호는 분명함
- 분기 유동비율 모델만 직전분기 기준선을 소폭 상회: **0.818 vs 0.793**
- SHAP beeswarm에서 기업 ID 영향이 거시변수보다 큼

# 3. 에이전트와 대시보드

- 도구 4종
  - 현재 위험 조회
  - 동종기업 건전성 순위
  - 관측범위 내 거시변수 민감도
  - Z-score SHAP 설명
- Gemini 2.5 Flash 자연어 질의와 5단 리포트
- FastAPI `/analyze`, `/report`
- Streamlit 6개 탭과 Markdown·PDF 출력
- 자동화 테스트 **7/7 통과**

## 데모 순서

현대모비스 선택 → 추이 → 시나리오 → 동종·SHAP → 챗봇 → PDF 리포트

# 4. 결론과 활용 범위

## 현재 가능한 것

- 21개사의 재무 위험 현황과 추이 점검
- 동종기업 대비 상대적 위치 확인
- 모델이 어떤 입력에 반응했는지 설명

- 신규기업과 관측범위 밖 외삽
- 투자 또는 신용 의사결정 자동화
